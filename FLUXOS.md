# Fluxos Operacionais

## Orçamento aprovado
1. Orçamento muda para aprovado.
2. Cria OS na Agenda.
3. Vincula cliente e veículo.
4. Mantém fotos e histórico.

## OS entregue
1. Agenda marca OS como Entregue.
2. Cria lançamento financeiro a receber.
3. Atualiza histórico do cliente.
4. Alimenta Relatórios.

## Financeiro pago
1. Lançamento muda para pago.
2. Atualiza caixa.
3. Atualiza metas.
4. Atualiza relatórios.
