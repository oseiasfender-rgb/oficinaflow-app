# Deploy OficinaOS

## GitHub Pages
1. Criar repositório.
2. Enviar `index.html`, `core.js`, `backup.js`, `manifest.json`, `sw.js`.
3. Ativar Pages em Settings → Pages.
4. Testar URL pública.

## Vercel
1. Importar repositório.
2. Framework: Other.
3. Build: vazio.
4. Output: raiz.
