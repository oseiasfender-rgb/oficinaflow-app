# OficinaOS V1 — MOTOR DE CONTAS REAL

## Regra definitiva

Conta criada, vencida ou pendente não entra no caixa.

A conta só vira despesa de caixa quando marcada como paga.

## Incluído
- `OficinaOS.contasResumo(mes)`
- `OficinaOS.pagarConta(id, dataPagamento)`
- `OficinaOS.desfazerPagamentoConta(id)`
- `OficinaOS.gerarRecorrenteProximoMes(conta)`
- deduplicação por `contaId`
- ações Pagar/Desfazer na aba Contas do Financeiro
