# OficinaOS V1 — MOTOR DE RECORRENTES REAL

## Regra
Recorrência gera contas futuras, mas não gera despesa de caixa automaticamente.

A despesa só entra no caixa quando a conta recorrente for marcada como paga.

## Incluído
- `OficinaOS.recorrentesResumo()`
- `OficinaOS.gerarRecorrentesAte(mesFinal)`
- `OficinaOS.criarRecorrenteManual(dados)`
- `OficinaOS.excluirRecorrenciaFutura(recurKey, apartirMes)`
- aba Recorrentes no Financeiro
- geração sem duplicar competência já existente
