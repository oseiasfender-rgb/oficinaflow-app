# Primeira ação real recomendada

Depois de abrir o sistema:

1. Rode no console:

```js
CleanStart.verifyZero()
```

2. Se `zero: true`, escolha um caminho:

## Caminho A — começar manualmente

- Configurar dados da empresa.
- Cadastrar primeiro cliente real.
- Cadastrar primeiro veículo real.
- Criar primeiro orçamento real.
- Exportar backup inicial.

## Caminho B — importar dados reais

- Usar `BackupCore.importFullJSON(...)` ou função de importação.
- Conferir clientes, orçamentos, agenda e financeiro.
- Exportar novo backup após a importação.

Não use dados de teste nesta versão.
