# OficinaOS — Pacote para Teste Real no GitHub Pages

## Status
Pacote baseado na RC2 corrigida para teste real controlado.

## Como subir
1. Crie ou abra o repositório no GitHub.
2. Envie todos os arquivos desta pasta para a raiz do repositório.
3. Em Settings > Pages, selecione branch `main` e pasta `/root`.
4. Abra o link público gerado pelo GitHub Pages.

## Teste mínimo obrigatório
1. Abrir no celular Android e no computador.
2. Criar primeiro cliente real.
3. Criar primeiro orçamento real.
4. Aprovar orçamento e conferir se gera OS na Agenda.
5. Marcar OS como entregue e conferir Receita no Financeiro.
6. Criar conta a pagar e marcar como paga; conferir Despesa.
7. Conferir Relatórios e Metas.
8. Gerar PDF e conferir valores/layout A4.
9. Exportar backup JSON.
10. Fechar, reabrir e conferir persistência.
11. Instalar como PWA no Android e abrir offline.

## Observação técnica
Ainda não é produção final. É pacote para teste real. Só deve virar versão final após passar no fluxo completo acima.
