# OFICINAOS V548.10 — REAL OPERATION TEST

## Objetivo
Validar o OficinaOS com operação real, sem dados fictícios e sem simulação artificial.

## Sequência obrigatória

1. Abrir `index.html`.
2. Configurar empresa.
3. Cadastrar cliente real.
4. Cadastrar veículo real.
5. Criar orçamento real.
6. Inserir materiais reais.
7. Inserir peças recuperadas, se houver.
8. Gerar PDF.
9. Criar OS na agenda.
10. Mover OS por etapas.
11. Lançar receita.
12. Lançar despesa.
13. Validar metas.
14. Validar relatórios.
15. Exportar backup.
16. Limpar sistema.
17. Importar backup.
18. Conferir se os dados voltaram.
19. Abrir IA e registrar consulta.
20. Testar instalação PWA.

## Regra
Se uma função falhar, registrar em `real-operation-test.html`.
Não corrigir no escuro.
