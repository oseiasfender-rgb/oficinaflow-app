# OficinaOS V550.60 — GitHub Pages / PWA Readiness

## Publicação rápida

1. Envie todos os arquivos deste pacote para a raiz do repositório.
2. No GitHub: Settings → Pages.
3. Source: Deploy from a branch.
4. Branch: `main` / folder: `/root`.
5. Aguarde o link do GitHub Pages.
6. Abra no Chrome Android e use “Adicionar à tela inicial”.
