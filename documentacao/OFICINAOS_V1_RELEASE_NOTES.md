# OFICINAOS V1 — Release Notes

O OficinaOS V1 consolida o ciclo V547 como sistema local definitivo para operação diária.

## Decisão final
- Visual Heritage.
- Motor V547.
- IndexedDB.
- Empresa configurável.
- Backup real importado no navegador.
- Evolução SaaS deixada para V2/V3.

## Não embutir backup real
O sistema deve permanecer limpo. O backup real deve ser importado após publicação ou teste local.
