OficinaOS V546.23 RC2 — STATE Único sem iframe

Arquitetura modular estável. Sem iframe, sem web-sandbox, sem bloqueio de rede. As integrações acontecem pelo STATE OficinaOS entre as páginas.

Decisão técnica: o painel unificado por iframe foi cancelado. Esta versão usa abertura direta dos módulos e integração por STATE único.

Fluxo operacional integrado

1. Clientes → OrçamentoAbra Clientes e use o botão da ponte STATE para enviar cliente/veículo ao orçamento.
2. Orçamento → FinanceiroAprovar cria receita pendente real.
3. Orçamento → AgendaAprovar também cria OS/agendamento.
4. Agenda → FinanceiroEntregar confirma recebimento ou mantém a receber.
5. Agenda → ClientesFotos, etapas e histórico ficam no STATE.
6. Relatórios / Metas / IALêem resumo real do STATE.
